# Favorite Movies

## Sobre o Projeto

Este projeto está sendo desenvolvido com o objetivo de praticar e consolidar conhecimentos em desenvolvimento backend utilizando Java e Spring Boot. Trata-se de uma API para gerenciamento de filmes, criada como exercício prático e também como parte de um portfólio para demonstrar evolução técnica e aplicação de boas práticas no desenvolvimento de software.

## Tecnologias Utilizadas

- Java  
- Spring Boot  
- Spring Data JPA  
- PostgreSQL  
- Maven
